### 这是一个基于DM H7系列开发板的仓库




#### 第一部分  **Git 指令大全**

##### 参考《Github入门与实践》

###### 1.```git init```  

初始化仓库 （在第一次新建本地库时使用） 使用后会在当前目录内容生成一个.git的文件夹来存储当前的仓库数据
###### 2.```git status```

用于显示当前Git仓库的过程，比如
```On branch main  //表示当前仓库处在main分支下 ```
```nothing to commit , working tree clean // 表明当前所有更改都已经提交 ```

###### 3. ```git add```

用于将文件加入暂存区，成为Git仓库的管理对象。
暂存区：提交之前的临时区域
